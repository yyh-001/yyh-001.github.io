手机开启热点

  1. 手机开启热点并打开clash
  2. clash中设置-覆写-启用允许来自互联网的连接
  3. 确保手机可以访问github
  4. jetson设备连接手机热点

jetson设置代理

查询手机ip地址

设置-我的设备-全部参数-状态信息-IP地址（每个手机不同）
我这里是172.18.0.1，后续所有要替换成你自己的IP地址

设置反向代理

设置-Network-Network Porxy-改为Manual
HTTP Proxy-172.18.0.1-7890
HTTPS Proxy-172.18.0.1-7890
Socks Host-172.18.0.1-7891
设置完浏览器就可以访问github了，但是终端还是无法访问

设置全局代理

sudo vi ~/.bashrc

在最后一行添加如下内容，记得替换成你手机的IP

function proxy() {
    export http_proxy=http://172.18.0.1:7890
    export https_proxy=$http_proxy
    echo -e "proxy on!"
}
function unproxy(){
    unset http_proxy https_proxy
    echo -e "proxy off!"
}

esc+:wq保存并退出，终端输入

source ~/.bashrc

打开代理

proxy()

显示proxy on! ，说明代理成功，如果后续需要关闭只需要输入unproxy()
现在你就可以访问github了

仅设置github全局代理

记得替换成你手机的IP

git config --global http.proxy http://172.18.0.1:7890
git config --global https.proxy http://172.18.0.1:7890
